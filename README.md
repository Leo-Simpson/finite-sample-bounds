# finite-sample-bounds

This repository contains the code associated to the (future) paper [*Finite-sample bounds for multi-output system identification*](https://arxiv.org/abs/2603.19073) by Léo Simpson, Katrin Baumgartner, Johannes Kohler and Moritz Diehl (pre-print).

The code runs the experiments described in Section V of the mentioned paper, and generate the four figures from there.
